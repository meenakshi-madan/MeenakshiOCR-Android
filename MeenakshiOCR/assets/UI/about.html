Version: 1.0

Developer: Meenakshi Madan
Developed as my final year project at UPG College of Management

Main purpose of this app is to represent text on an image as plain text. The appl does this by (optionally) pre-processing a selected image, and performing Optical Character Recognition on it.
The app provides several ways to use this text:-

  • Copy to clipboard
  • Save as text file (.txt)
  • Google the text (opens web browser)
  • Share with other apps
  • Directly send an SMS


The application also locally stores the history of the 5 most recent OCR'ed text. This can be accessed on History screen.


Customize this app on the Settings screen.


To find out more about what the icons and buttons do, for a quick tutorial, or other information, go back to the Help Screen.
